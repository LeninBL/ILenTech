Resumen: HTML y XML

Construcción de página web con HTML y XML

HTML estructura la página web con etiquetas semánticas, mientras XML define datos estructurados, usado para intercambio y configuración.

Estructura y etiquetas principales de HTML5

  • <!DOCTYPE html> para declarar el tipo de documento.
  • <html>, <head>, <body>: bloques base.
  • <header>, <nav>, <main>, <footer>: etiquetas semánticas.
  • <p>, <a>, <img>, <section>, <article>, <div>.

Árbol DOM: nodos, renderizado y ciclo de vida

  • DOM es la representación en árbol de la página (nodos, elementos, texto).
  • El navegador parsea el HTML, genera el DOM y lo combina con CSS para renderizar.
  • Eventos: carga, DOMContentLoaded, interacción, actualización dinámica.

Optimización SEO y compatibilidad entre navegadores

  • Uso correcto de etiquetas semánticas para mejorar SEO.
  • Metadatos como <meta> y <title>.
  • Pruebas en diferentes navegadores y uso de polyfills.
  • Carga rápida y accesibilidad para usuarios y motores de búsqueda.
