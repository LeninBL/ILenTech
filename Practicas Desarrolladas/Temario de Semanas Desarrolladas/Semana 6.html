Framework JS - Fundamentos

Componentes, props y estilización

Conceptos Clave

Estructura Básica

  • Client-side rendering: Renderizado en el navegador
  • Gestión de dependencias: npm/yarn/pnpm
  • Herramientas: Vite, Webpack, Parcel
  • Componentes: Bloques reutilizables

Estilización

  • Inline styles: Objetos de estilos en JS
  • CSS Modules: Estilos con scope local
  • Styled Components: CSS-in-JS
  • Tailwind: Clases utilitarias

Ejemplos Prácticos

1. Componente con Props

Simulación de un componente React que acepta props.

function Saludo({ nombre, color }) {
  return (
    <div style={{ color }}>
      Hola, {nombre}!
    </div>
  );
}

// Uso:
<Saludo nombre="Ana" color="blue" />

2. Componente con Children

Simulación de un componente que acepta contenido children.

function Tarjeta({ children, titulo }) {
  return (
    <div className="tarjeta">
      <h3>{titulo}</h3>
      <div className="contenido">
        {children}
      </div>
    </div>
  );
}

// Uso:
<Tarjeta titulo="Ejemplo">
  <p>Contenido children</p>
</Tarjeta>

Durante esta práctica he comprendido que:

  • Los componentes son la unidad fundamental de los frameworks modernos.
  • Las props permiten la comunicación entre componentes padre-hijo.
  • Los children ofrecen una forma poderosa de composición.
  • Cada método de estilización tiene sus casos de uso ideales.

Aplicación en proyectos reales:

Usaré componentes para crear interfaces modulares, props para configuración y children para contenido dinámico. Evaluaré el método de estilización según las necesidades de cada proyecto.
