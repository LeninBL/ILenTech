2. Funciones en JS

Comparación entre función tradicional y arrow function.

Función Tradicional

function suma(a, b) {
  return a + b;
}

Arrow Function

const sumaArrow = (a, b) => a + b;

3. Animación con Canvas

Pelota rebotando usando JavaScript y el elemento canvas.

Reflexión Final

Aspecto JavaScript TypeScript
Flexibilidad Alta (tipado dinámico) Moderada (tipado estático)
Escalabilidad Desafíos en proyectos grandes Excelente para proyectos complejos

Durante esta práctica he aprendido que:

  • JavaScript es ideal para prototipado rápido y proyectos pequeños.
  • TypeScript añade seguridad de tipos que previene errores comunes.
  • El manejo del DOM con JavaScript es fundamental para interactividad web.
  • Canvas ofrece posibilidades creativas para gráficos y animaciones.

¿Cómo aplicar esto en el futuro?

Usaré TypeScript en proyectos profesionales para mayor robustez, y JavaScript para pequeños scripts y prototipos. Canvas será mi herramienta para visualizaciones dinámicas.
