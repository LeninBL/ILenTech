Resumen: HTML y XML

Construcción de página web con HTML y XML

HTML define la estructura y contenido visual de páginas web. XML se usa para almacenar y transportar datos estructurados, complementando la web.

Estructura y etiquetas principales de HTML5

  • <!DOCTYPE html> declara el documento.
  • <html>, <head>, <body> forman la base.
  • Etiquetas semánticas: <header>, <nav>, <main>, <footer>.
  • Elementos comunes: <p>, <a>, <img>, <section>, <article>.

Árbol DOM: nodos, renderizado y ciclo de vida

  • DOM es la representación en árbol del documento HTML.
  • El navegador crea el DOM al cargar la página y usa CSS para renderizar el contenido.
  • Eventos importantes: carga, DOMContentLoaded, interacción y actualizaciones dinámicas.

Optimización SEO y compatibilidad entre navegadores

  • Uso correcto de etiquetas semánticas para mejorar el posicionamiento.
  • Metadatos clave: <meta> y <title>.
  • Pruebas cruzadas y uso de polyfills para compatibilidad.
  • Optimización de carga y accesibilidad para usuarios y motores de búsqueda.
